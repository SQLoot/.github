# Support

## Getting Help

### Documentation

Check the README in each repository for project-specific documentation.

### Discussions

Use GitHub Discussions for:
- Questions about usage
- Feature requests
- General discussion

### Issues

Use GitHub Issues for:
- Bug reports
- Confirmed problems with reproduction steps

## Response Times

We're a small team and respond as quickly as possible. Please be patient.

| Type             | Expected Response  |
| ---------------- | ------------------ |
| Security issues  | < 48 hours         |
| Bug reports      | < 7 days           |
| Feature requests | As capacity allows |
